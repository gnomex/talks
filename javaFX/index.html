JavaFX

O que é ?

JavaFX é uma plataforma de software para a criação e entrega de aplicações ricas para internet (RIAs) que podem ser executados em uma ampla variedade de dispositivos.

Escrito como uma API Java, o código escrito em JavaFX pode referênciar APIs de qualquer biblioteca Java.

Por exemplo, as aplicações JavaFX podem usar as Java APIs para acessar os recursos nativos do sistema e se conectar a aplicativos server-based middleware

Os estilos são customizaveis por Cascading Style Sheets (CSS), que separa estilo de implementação

Para que serve ?

Interfaces gráficas ricas

  • organização de código
  • manutenção rápida e descomplicada

e o principal motivo, qualidade gráfica para uma área onde os recursos de programação são limitados.

Sim, é possível criar aplicações Desktop com qualidade gráfica avançada, com conceitos CSS e belos efeitos visuais!

Por que é importante ?
  • Portável: multiplataforma
  • Responsivo
